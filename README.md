# 🏠 House Price Prediction Web Application

This project is a Machine Learning-based web application that predicts house prices based on property features provided by the user. The application is built using Flask and Scikit-Learn and allows users to get instant house price predictions through a simple and interactive web interface.

## 🌐 Live Demo

https://house-price-pridiction-production.up.railway.app

## 💻 GitHub Repository

https://github.com/Anilkumar7722/House-price-pridiction

## 🚀 Project Overview

The main goal of this project is to estimate house prices using machine learning techniques. Users can enter property details such as area, number of bedrooms, age of the property, garage capacity, pool availability, and gym facilities. The trained machine learning model then predicts the estimated house price.

This project helped me gain practical experience in machine learning, web development, model deployment, and version control.

## 📊 Project Workflow

1. Collect and prepare housing data
2. Train a Machine Learning model
3. Save the trained model using Joblib
4. Build a Flask web application
5. Connect the model with the web interface
6. Deploy the application online using Railway


## ⚙️ How to Run Locally

Clone the repository:

```bash
git clone https://github.com/Anilkumar7722/House-price-pridiction.git
```

Move to the project directory:

```bash
cd House-price-pridiction
```

Install required packages:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
